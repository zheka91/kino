# kino

КиноПросмотр

Сервис для запоминания фильмов и сериалов для будующих просмотров, а также отметки просмотренных.
Функционал доступен после регистрации, ну и конечно авторизации.

Используется самоподписанный SSL сертификат:
https://film.is-best.net/index.php
На всякий случай без него (могут не грузиться изображения):
http://film.is-best.net/index.php

Возможные доработки:
- Добавление друзей.
- Просмотр библиотеки друзей.
- Порекомендовать фильм другу.
